# Semiconductor Yield Analysis — SECOM Dataset

Real-world semiconductor manufacturing data analysis using the UCI SECOM dataset
(1,567 wafers, 590 sensor measurements per wafer from a wafer fab production line).

## Analysis performed
- Missing value detection and data cleaning (NaN handling)
- Yield % calculation: pass/fail wafer count and defect rate
- Parametric trend analysis: pass vs fail sensor value distributions
- Statistical process control (SPC) chart with UCL/LCL limits
- Outlier detection using z-score method
- Sensor correlation heatmap for yield engineering insights

## Tools used
Python | pandas | numpy | matplotlib | seaborn | scipy

## Dataset source
UCI Machine Learning Repository — SECOM Semiconductor Manufacturing Dataset
